¿Existen los condicionales en Sass?

¡Sí! Sass incluye condicionales a través de la directiva @if. Esto permite que puedas aplicar estilos dependiendo de una condición lógica.

Ejemplo de uso

Un caso común es cambiar colores dependiendo de una variable:


$theme: dark;

.button {
  @if $theme == light {
    background-color: #fff;
    color: #000;
  } @else {
    background-color: #000;
    color: #fff;
  }
}

¿Es recomendado?

Sí, pero con moderación. Es útil cuando tienes configuraciones dinámicas (por ejemplo, temas claros y oscuros). Sin embargo, no abuses de los condicionales, ya que podrían dificultar la legibilidad de tus estilos.

Cuándo usarlos

  • Para gestionar temas claros/oscuros.
  • Para definir estilos basados en configuraciones específicas.
  • Cuando necesites estilos reutilizables basados en condiciones.
